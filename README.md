# 1. 摘要資訊

* 以教育為目的開發的簡易靶機，包含滲透以及提權這兩個部分
* 相關的主題是利用sql injection作為提權的設定，以ruby檔讀取特定yaml檔的漏洞為題權的內容
* 同時利用docker幫助教育者能夠快速架設好所需的靶機環境
